+

NowView 是否支持所有网站？

+

NowView 支持大多数网站。但由于某些网站的安全策略（如 GitHub、银行网站），可能会阻止在 iframe 中显示。我们会持续优化兼容性。

我的数据会安全吗？

+

完全安全！NowView 不收集任何个人数据，所有设置都保存在本地。会员信息也仅存储在你的浏览器中，我们不上传任何用户数据。
