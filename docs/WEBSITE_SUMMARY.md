## 💡 本地开发

### 使用 Python 3 运行本地服务器
```bash
cd docs
python -m http.server 8000
# 打开 http://localhost:8000
```

### 使用 Node.js
```bash
npx http-server docs
```

### 使用 Ruby
```bash
cd docs
ruby -run -ehttpd . -p8000
```
