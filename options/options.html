关于 NoTab

版本：1.0.0

NoTab 是一个功能强大的链接预览工具，让您无需打开新标签页即可快速浏览网页内容。

核心功能

  • 🔗 链接预览 - 在当前页面直接预览链接
  • 📖 阅读模式 - 沉浸式阅读体验
  • 🎬 视频模式 - 专注观看视频内容
  • 🌐 快速翻译 - 一键翻译选中文本
  • 🔍 快速搜索 - 多搜索引擎支持
  • 🎨 多主题 - 亮色/暗色主题
  • ☁️ 云同步 - Google账号自动同步设置

技术栈

基于 Manifest V3 和 Vanilla JavaScript 开发
