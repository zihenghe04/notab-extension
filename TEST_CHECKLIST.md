# NoTab 插件测试清单

## 安装前准备

- [ ] 确认所有文件已创建
- [ ] 生成图标文件（使用 generate-icons.html）
- [ ] 将图标文件放入 icons/ 目录

## 安装测试

- [ ] 打开 chrome://extensions/
- [ ] 启用"开发者模式"
- [ ] 成功加载扩展
- [ ] 工具栏显示紫色N图标
- [ ] 无错误提示

## 功能测试

### 1. 链接预览功能
- [ ] **Ctrl+点击链接** - 打开预览窗口
- [ ] **拖拽链接** - 在页面空白处释放后打开预览
- [ ] **右键菜单** - "在预览窗口中打开"选项可用
- [ ] 预览窗口正确显示网页内容
- [ ] 可以在预览窗口中滚动和交互

### 2. 窗口操作
- [ ] **拖拽头部** - 可以移动预览窗口
- [ ] **调整大小** - 8个手柄都可以调整窗口大小
- [ ] **固定按钮** - 点击后窗口变为固定状态（显示蓝色边框）
- [ ] **关闭按钮** - 可以关闭预览窗口
- [ ] **新标签打开** - 点击🔗按钮在新标签打开

### 3. 多窗口管理
- [ ] 可以同时打开多个预览窗口
- [ ] 点击窗口会置顶显示
- [ ] Esc键关闭所有未固定窗口
- [ ] 固定的窗口不会被Esc关闭

### 4. 阅读模式
- [ ] 点击📖按钮激活阅读模式
- [ ] 成功提取文章标题和内容
- [ ] 显示清晰易读的排版
- [ ] 可以滚动阅读内容

### 5. 视频模式
- [ ] 在YouTube等视频网站打开预览
- [ ] 点击🎬按钮激活视频模式
- [ ] 视频正确显示和播放
- [ ] 全屏按钮可用

### 6. 翻译功能
- [ ] 选中文本后显示🌐按钮
- [ ] 点击翻译按钮显示翻译结果
- [ ] 中文翻译成英文
- [ ] 英文翻译成中文
- [ ] 翻译结果窗口可关闭

### 7. 搜索功能
- [ ] 选中文本后显示🔍按钮
- [ ] 点击搜索按钮在预览窗口显示搜索结果
- [ ] 可以选择不同搜索引擎（在设置中）

### 8. Popup界面
- [ ] 点击工具栏图标打开popup
- [ ] 显示当前预览统计（活动数、固定数）
- [ ] 级联布局按钮可用
- [ ] 平铺布局按钮可用
- [ ] 关闭所有按钮可用
- [ ] 设置开关可以切换（拖拽、Ctrl+点击、悬停）
- [ ] 主题切换按钮可用

### 9. 设置页面
- [ ] 右键图标选择"选项"打开设置页面
- [ ] 可以切换标签页
- [ ] 所有设置项都可以修改
- [ ] 点击"保存设置"后显示成功提示
- [ ] 设置会自动同步（在多台设备测试）
- [ ] 重置按钮可以恢复默认设置

### 10. 主题功能
- [ ] 切换到暗色主题后界面变暗
- [ ] 切换到亮色主题后界面变亮
- [ ] 自动模式根据系统设置切换

### 11. 快捷键
- [ ] `Esc` - 关闭未固定窗口
- [ ] `Ctrl+Shift+C` - 级联布局
- [ ] `Ctrl+Shift+T` - 平铺布局
- [ ] `Ctrl+Shift+X` - 关闭所有窗口

## 边界情况测试

- [ ] 访问HTTPS和HTTP网站都能预览
- [ ] 在Chrome商店等特殊页面不会报错
- [ ] 同时打开超过5个窗口会自动关闭最旧的
- [ ] 网络断开时翻译显示错误提示
- [ ] CORS限制的网站显示提示信息

## 性能测试

- [ ] 打开预览窗口响应速度快（<1秒）
- [ ] 多个窗口同时存在不卡顿
- [ ] 拖拽和调整大小流畅
- [ ] 内存占用合理

## 已知限制

以下是预期的限制（不算bug）：
- Chrome商店页面无法注入内容脚本
- 某些设置了X-Frame-Options的网站无法在iframe中显示
- 跨域iframe无法执行阅读模式（会尝试fetch方案）
