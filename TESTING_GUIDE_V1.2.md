# NowView v1.2 测试指南

## 🎯 新增功能测试清单

### 1️⃣ 会员系统测试

#### Free 用户测试
- [ ] 打开 Popup，查看会员徽章显示 `✨ Free`
- [ ] 查看升级卡片显示 "今日还剩 X/10 次预览"
- [ ] 创建 10 个预览，观察次数递减
- [ ] 第 10 次预览后，再次尝试预览
- [ ] 应该看到限制提示框：
  - ⚠️ 图标
  - "今日预览次数已用完，升级 Pro 解锁无限预览！"
  - "🎫 升级 Pro" 按钮
  - 关闭按钮
- [ ] 点击 "升级 Pro" 按钮应该打开扩展 Popup

#### Pro 用户测试
- [ ] 点击 Popup 中的 "🎫 输入兑换码" 按钮
- [ ] 输入有效兑换码（如：`NOWVIEW-PRO-2024`）
- [ ] 点击 "激活" 按钮
- [ ] 应该看到成功提示：🎉 Pro 会员激活成功
- [ ] 会员徽章变为 `👑 Pro`
- [ ] 升级卡片消失
- [ ] 可以无限制创建预览（测试创建 15+ 个预览）
- [ ] Popup 重新打开，状态依然是 Pro
